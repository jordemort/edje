Set Edje color class. 

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em color\_\-class}]\item[{\em r}]Object Red value \item[{\em g}]Object Green value \item[{\em b}]Object Blue value \item[{\em a}]Object Alpha value \item[{\em r2}]Outline Red value \item[{\em g2}]Outline Green value \item[{\em b2}]Outline Blue value \item[{\em a2}]Outline Alpha value \item[{\em r3}]Shadow Red value \item[{\em g3}]Shadow Green value \item[{\em b3}]Shadow Blue value \item[{\em a3}]Shadow Alpha value\end{description}
\end{Desc}
Sets the color values for a process level color class. This will cause all edje parts in the current process that have the specified color class to have their colors multiplied by these values. (Object level color classes set by \hyperlink{edje_8c_f7ab2c66bb1793bf26d56d322abb9fee}{edje\_\-object\_\-color\_\-class\_\-set()} will override the values set by this function).

Add a callback for a signal emitted by {\em obj\/}. 

\begin{Desc}
\item[Parameters:]
\begin{description}
\item[{\em obj}]A valid Evas\_\-Object handle \item[{\em emission}]The signal name \item[{\em source}]The signal source \item[{\em func}]The callback function to be executed when the signal is emitted \item[{\em data}]A pointer to data to pass in to the callback function\end{description}
\end{Desc}
Connects a callback function to a signal emitted by {\em obj\/}. In EDC, a program can emit a signal as follows:



\begin{Code}\begin{verbatim} program {
   name: "emit_example";
   action: SIGNAL_EMIT "a_signal" "a_source";
 }
\end{verbatim}
\end{Code}



Assuming a function with the following declaration is definded:



\begin{Code}\begin{verbatim} void cb_signal(void *data, Evas_Object *o, const char *emission, const char *source);
\end{verbatim}
\end{Code}



a callback is attached using:



\begin{Code}\begin{verbatim} edje_object_callback_add(obj, "a_signal", "a_source", cb_signal, data);
\end{verbatim}
\end{Code}



Here, {\em data\/} is an arbitrary pointer to be used as desired. Note that {\em emission\/} and {\em source\/} correspond respectively to first and second parameters to the SIGNAL\_\-EMIT action.

Internal edje signals can also be attached to, and globs can be in either the emission or source name. e.g.



\begin{Code}\begin{verbatim} edje_object_callback_add(obj, "mouse,down,*", "button.*", NULL);
\end{verbatim}
